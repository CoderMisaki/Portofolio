Design Philosophy

This portfolio must feel like a modern digital creative space.

The overall experience should combine:

- creative energy
- premium startup aesthetics
- playful interactions
- clean professional structure
- modern storytelling

The UI should feel:

- human
- warm
- artistic
- interactive
- premium
- memorable

Avoid generic AI-generated portfolio layouts.

The experience should feel inspired by:

- Apple
- Linear
- Figma
- modern creative studios
- interactive startup landing pages

---

Main Concept

1. Creative Garden

Theme:
A modern digital garden.

The interface should feel organic, alive, soft, and creative.

Visual Elements

- subtle floating leaves
- organic doodle lines
- hand-drawn accents
- floating plant decorations
- sticky-note inspired cards
- soft gradients
- curved organic shapes
- paper-like layered cards

Mood

- calming
- creative
- approachable
- elegant
- modern

Decorative Rules

- decorations must be subtle
- never clutter the UI
- use whitespace generously
- decorations should guide attention naturally

---

2. Playful Professional

The website must balance:

- professionalism
- creativity
- startup aesthetics
- playful interactions

Characteristics

- large typography
- strong visual hierarchy
- soft rounded corners
- smooth hover interactions
- premium spacing
- clean layouts
- elegant cards
- modern buttons
- interactive micro animations

Layout Direction

- asymmetric layouts allowed
- zig-zag sections preferred
- alternating visual rhythm between sections
- avoid rigid grid-only feeling

---

3. Digital Creator Desk

The portfolio should feel like a creator workspace.

Optional Decorative Components

- sticky notes
- fake sketches
- mini code previews
- github contribution widgets
- currently building section
- spotify widget
- creator desk elements
- floating desktop objects

Important

These elements should support the storytelling.
Do not overload the UI.

Color Rules

Background

Use:

- soft milk white backgrounds
- subtle warm gradients
- avoid pure white (#FFFFFF)

Accent Usage

Green:

- primary CTA
- hover glow
- active states

Yellow:

- highlights
- decorative accents
- playful emphasis

Orange:

- energetic accents
- icon highlights
- interactive moments

Avoid

- neon colors
- oversaturated palettes
- harsh contrast
- dark heavy backgrounds

Icons

Animations

Animation Philosophy

Animations should feel:

- premium
- smooth
- subtle
- responsive
- intentional

Avoid:

- excessive motion
- flashy transitions
- heavy parallax
- distracting effects

---

Preferred Animations

Required

- smooth scrolling
- reveal animations
- floating blobs
- magnetic buttons
- card hover tilt
- soft fade transitions
- lightweight parallax
- cursor glow effect

Decorative Motion

- floating leaves
- doodle movement
- soft idle animations

---

Performance Rules

Animations must:

- remain lightweight
- maintain smooth FPS
- work well on mobile
- avoid excessive GPU usage

Do not use:

- massive particle systems
- over-engineered WebGL effects
- excessive blur rendering
