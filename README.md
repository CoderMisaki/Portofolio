# Portfolio - Muhammad Fadillah

## Project Description
Creative Garden themed personal portfolio website showcasing web development projects, skills, and contact information.

## Run Locally
1. Install dependencies:
   ```bash
   npm install
   ```
2. Start local server:
   ```bash
   npm run dev
   ```

## Build
```bash
npm run build
```

## Deploy
- Recommended: Vercel deployment from this repository.
- Ensure `vercel.json` headers are applied in production.

## Production Checklist
- [ ] Build succeeds.
- [ ] No console errors in browser.
- [ ] External links with `target="_blank"` include `rel="noreferrer noopener nofollow"`.
- [ ] Project iframes use `loading="lazy"`, `sandbox`, and `referrerpolicy`.
- [ ] SEO meta tags and structured data are present.
- [ ] Responsive layout verified on mobile and desktop.
